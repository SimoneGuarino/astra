# astra
Astra is a local-first desktop AI companion built with Tauri v2, Rust, React, Ollama, and dedicated Python speech workers. It features streaming conversation, continuous voice mode, wake-based activation, barge-in interruption, local speech-to-text and text-to-speech pipelines, request lifecycle management, metrics, audio queueing, and a sci-fi-inspired overlay UI. Astra is designed with a modular, production-minded architecture to evolve into a powerful personal assistant for coding, productivity, desktop automation, and natural voice interaction.
